DoH 和 DoT

基本概念

DNS(Domain Name Server)

DoH(DNS over HTTPS) 基于 HTTPS 和 HTTP2，使用端口443。协议标准 RFC 7858、RFC 8310

DoT(DNS over TLS) 基于 TCP，使用端口853。协议标准 RFC 8484

DoH 和 DoT 均起到了加密 DNS 请求的作用。
